\chapter{List of additional files in~electronic submission}

The electronic submission attached to this thesis contains a compressed archive (\texttt{.zip}) organized as follows:

\begin{itemize}
    \item \textbf{Source Code:} The complete source code of the system, divided into:
    \begin{itemize}
        \item \texttt{/pc\_server} -- Python scripts for the Vision and Telemetry modules (\texttt{bilard.py}, \texttt{sensors.py}) including the \texttt{req.txt} file.
        \item \texttt{/android\_app} -- The Android Studio project files for the mobile sensor application.
    \end{itemize}
    
    \item \textbf{Executables:}
    \begin{itemize}
        \item \texttt{SensorApp.apk} -- The compiled installation package for the Android mobile device.
    \end{itemize}
    
    \item \textbf{Configuration:}
    \begin{itemize}
        \item \texttt{config.json} -- Default configuration file with calibration parameters and anthropometric data structure.
    \end{itemize}
    
    \item \textbf{Documentation:}
    \begin{itemize}
        \item \texttt{README.md} -- Short instruction on how to run the environment and install dependencies.
    \end{itemize}
\end{itemize}
